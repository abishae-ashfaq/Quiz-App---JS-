# GitHub & JavaScript Quiz App 

Description:
A web-based multiple-choice quiz testing users on GitHub and JavaScript. Built with HTML, CSS, JavaScript, and Bootstrap.

**Features**

10 interactive multiple-choice questions

Navigate with Back and Next buttons

Retains selected answers

Submit button shows final score

Responsive design for mobile & desktop

Technologies:
HTML5 | CSS3 | JavaScript | Bootstrap 5

Extras:
Easily extensible, modular code, and clean, user-friendly interface.
